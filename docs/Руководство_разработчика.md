# Руководство разработчика

## Состав проекта

- `src/main.ts` - исходный код;
- `main.js` - собранный runtime-файл;
- `styles.css` - стили интерфейса;
- `manifest.json` - манифест Obsidian;
- `versions.json` - версия плагина;
- `package.json`, `tsconfig.json`, `esbuild.config.mjs` - файлы сборки;
- `docs/` - документация пользователя и разработчика.

## Ключевые функции

- `onload` - загрузка данных, регистрация команд и запуск Focus Mode после включения плагина;
- `enableFocusMode` - включает режим концентрации;
- `disableFocusMode` - выключает только режим концентрации;
- `exitAndDisablePlugin` - возвращает интерфейс и выключает плагин в `Community plugins`;
- `startTimer`, `pauseTimer`, `resetTimer`, `switchMode` - управление таймером;
- `startPanelDrag` - перетаскивание панели;
- `applyPanelPosition` - восстановление сохранённой позиции панели.

## Хранение данных

Плагин сохраняет настройки и состояние через стандартные методы Obsidian `saveData` и `loadData`.

Сохраняются:

- длительность фокуса;
- длительность перерыва;
- режим автозапуска;
- состояние таймера;
- количество завершённых фокус-сессий;
- положение панели.

## Проверка

```bash
node --check main.js
```

Затем нужно установить плагин в тестовый vault и проверить:

1. включение через `Community plugins`;
2. автоматическое появление панели;
3. скрытие интерфейса;
4. работу таймера;
5. перетаскивание панели;
6. отключение через кнопку `Выход`.
