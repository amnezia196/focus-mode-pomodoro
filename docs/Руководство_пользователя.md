## 1. Назначение плагина

Focus Mode Pomodoro - это плагин для Obsidian, который помогает работать с заметками без лишних отвлекающих элементов. При включении плагина запускается режим концентрации: скрываются боковые панели, ribbon, вкладки, заголовок заметки и строка состояния.

На экране остаётся рабочая область Obsidian и компактная панель Pomodoro-таймера. Панель можно перемещать, задавать своё время фокуса и перерыва, запускать таймер, ставить его на паузу и завершать работу одной кнопкой.

## 2. Что входит в продукт

- EXE-установщик `FocusModePomodoroSetup.exe`.
- Runtime zip для ручной установки.
- Исходный код плагина в папке репозитория.
- Документация пользователя и разработчика.
- Отчёт и презентация проекта.

## 3. Требования

- Установленный Obsidian.
- Открытый Obsidian vault, в который устанавливается плагин.
- Разрешённые сторонние плагины в `Settings -> Community plugins`.
- Windows для запуска EXE-установщика. Ручная установка работает через копирование файлов плагина.

## 4. Установка через EXE

1. Откройте папку `05_Установщик`.
2. Запустите `FocusModePomodoroSetup.exe`.
3. Если Windows покажет предупреждение о неизвестном издателе, подтвердите запуск.
4. Выберите папку Obsidian vault. Нужно выбрать именно папку vault, внутри которой есть `.obsidian`.
5. Дождитесь завершения установки.
6. Откройте Obsidian и перейдите в `Settings -> Community plugins`.
7. Включите `Focus Mode Pomodoro`.

**Важно:** После включения плагина панель Focus Mode появляется автоматически. Отдельно искать команду запуска не нужно.

## 5. Ручная установка

1. Откройте папку Obsidian vault в проводнике.
2. Перейдите в `.obsidian/plugins`.
3. Создайте папку `focus-mode-pomodoro`.
4. Скопируйте в неё файлы `manifest.json`, `main.js`, `styles.css` из runtime zip или из `dist/focus-mode-pomodoro`.
5. Перезапустите Obsidian.
6. Включите плагин в `Settings -> Community plugins`.

## 6. Первый запуск

После включения в `Community plugins` плагин сразу запускает Focus Mode. Интерфейс Obsidian становится минималистичным, а на экране появляется панель таймера.

Если плагин уже включён, режим концентрации можно переключать иконкой таймера в левой панели Obsidian.

## 7. Панель Focus Mode

| Элемент | Назначение |
| --- | --- |
| Фокус / Перерыв | Показывает текущий режим таймера. |
| Поле минут | Позволяет задать своё время фокуса или перерыва. |
| Большой таймер | Показывает оставшееся время. |
| Прогресс-бар | Показывает ход текущей сессии. |
| Старт / Пауза | Запускает или останавливает отсчёт. |
| Сброс | Возвращает время текущего режима к началу. |
| Перерыв / Фокус | Переключает режим таймера. |
| Выход | Возвращает обычный интерфейс и выключает плагин в настройках. |

## 8. Как задать своё время

1. Найдите поле минут рядом с меткой `Фокус` или `Перерыв`.
2. Введите нужное число минут.
3. Нажмите `Enter` или кликните вне поля.
4. Таймер сразу перестроится под новое значение.

**Важно:** Для режима `Фокус` доступен диапазон от 1 до 120 минут. Для режима `Перерыв` - от 1 до 60 минут.

## 9. Как переместить панель

1. Наведите мышь на верхнюю строку панели, где написано `Focus Mode`.
2. Зажмите левую кнопку мыши.
3. Перетащите панель в удобное место.
4. Отпустите кнопку мыши.

**Важно:** Положение панели сохраняется. При следующем запуске она появится там, куда её переместили.

## 10. Выход и отключение

Кнопка `Выход` завершает Focus Mode, возвращает обычный интерфейс Obsidian и выключает сам плагин в `Community plugins`.

Если плагин выключен, на экране не остаётся его панели, статуса или других элементов. Чтобы снова начать работу, включите плагин в настройках Obsidian.

## 11. Частые проблемы

| Проблема | Что сделать |
| --- | --- |
| Плагин не появился в списке | Проверьте, что файлы лежат в `.obsidian/plugins/focus-mode-pomodoro`. |
| Установщик просит выбрать папку | Выберите папку vault, а не `.obsidian` и не диск `C:\`. |
| Панель мешает заметке | Перетащите её за верхнюю строку `Focus Mode`. |
| После выхода плагин выключился | Это нормальное поведение. Снова включите его в `Community plugins`. |
| Не меняется время | Введите число минут и нажмите `Enter`. |

## 12. Что показать при проверке

1. Включить плагин в `Community plugins`.
2. Показать автоматическое появление панели.
3. Показать скрытие отвлекающих элементов интерфейса.
4. Запустить, поставить на паузу и сбросить таймер.
5. Изменить время в поле минут.
6. Перетащить панель.
7. Переключить `Фокус` и `Перерыв`.
8. Нажать `Выход` и показать, что плагин выключился.
