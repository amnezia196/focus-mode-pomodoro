# Focus Mode Pomodoro

Плагин для Obsidian, который включает режим концентрации и добавляет Pomodoro-таймер прямо в рабочее окно.

Автор: Емешкин Сергей Сергеевич, РИЗ-150045д, ИТОО УрФУ.

## Возможности

- скрытие отвлекающих элементов Obsidian;
- Pomodoro-таймер с режимами фокуса и перерыва;
- настройка времени прямо в панели;
- запуск, пауза и сброс таймера;
- переключение между фокусом и перерывом;
- перетаскивание панели и сохранение ее положения;
- выход из режима концентрации с отключением плагина в настройках Obsidian.

## Структура репозитория

- `src/main.ts` - исходный TypeScript-код плагина;
- `main.js` - собранный файл плагина для Obsidian;
- `styles.css` - стили панели и режима концентрации;
- `manifest.json` - манифест Obsidian-плагина;
- `versions.json` - версия плагина для Obsidian;
- `package.json`, `tsconfig.json`, `esbuild.config.mjs` - файлы сборки;
- `docs/Руководство_пользователя.md` - руководство пользователя;
- `docs/Руководство_разработчика.md` - руководство разработчика.

## Ручная установка

1. Открыть папку Obsidian vault.
2. Перейти в `.obsidian/plugins/`.
3. Создать папку `focus-mode-pomodoro`.
4. Скопировать туда `manifest.json`, `main.js`, `styles.css`.
5. Открыть Obsidian.
6. Включить `Focus Mode Pomodoro` в `Settings -> Community plugins`.

## Проверка кода

```bash
npm install
npm run check
```

Для проверки в Obsidian достаточно установить три runtime-файла `manifest.json`, `main.js`, `styles.css` в vault и включить плагин.
